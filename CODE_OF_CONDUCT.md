# Code of Conduct

## Our Standard

We want Localzet Deployer to be a professional and respectful project for developers and operators from different backgrounds.

Examples of expected behavior:

- be respectful and precise
- assume good faith
- give actionable technical feedback
- discuss ideas without personal attacks

Examples of unacceptable behavior:

- harassment or discrimination
- abusive or insulting language
- bad-faith disruption
- publishing private information without permission

## Enforcement

Project moderation is maintained by Ivan Zorin.

If you need to report conduct issues, contact:

- [creator@localzet.com](mailto:creator@localzet.com)

## Scope

This code of conduct applies to project spaces, issues, pull requests, discussions, and other public interactions around the project.
